{% trans "Client applications" %}

{% blocktrans %} This web service offers a public API that can be used to query the podcast directory and an authenticated API for synchronizing subscription lists and episode actions between the web service and multiple clients. {% endblocktrans %}

{% trans "Applications" %}

  • gPodder: {% blocktrans %} An open source podcast client for Linux, Windows, Maemo and Mac OS X. Support for gpodder.net is directly built into gPodder. {% endblocktrans %}
  • BashPodder: {% blocktrans %} A simple command-line podcatcher. To synchronize your bp.conf file with gpodder.net, use the "bpsync" utility included in mygpoclient (see below). {% endblocktrans %}

{% trans "Libraries" %}

  • mygpoclient: {% blocktrans %} This is the official reference implementation of the client-side API interface. mygpoclient is written in Python and is used by gPodder to interface with the web service. {% endblocktrans %}
  • mygpodderlib: {% blocktrans %} A Java library by Nico Heid for use on Android or other Java-based platforms - under development. {% endblocktrans %}
